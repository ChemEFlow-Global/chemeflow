ChemEFlow v8.0 PRO 🧪⚙️🏗️⚡🏭

**Engineering Intelligence Platform — Integrated Engineering Analysis & Process Simulation OS**

ChemEFlow is a futuristic, installable Progressive Web App (PWA) built for 
multi-disciplinary engineering professionals and students. It features a 
tactical HUD interface, 100+ professional calculators, real-time data 
visualization, and full offline support.

---

## 🌟 Key Features

**Multi-Branch Engineering:** Dedicated calculator modules for Chemical, 
Mechanical, Civil, Electrical, and Industrial Engineering — 100+ tools total.

**Carbon Capture & Sustainability (CCU):** Specialized modules for CO₂ 
absorption rate, solvent efficiency, CCS energy penalty, biomass conversion 
yield, and bio-leather (bacterial cellulose) yield calculations.

**Live Data Charts:** Dynamic Chart.js visualizations render automatically 
after CCU and key calculations — line graphs, bar charts, mass balance diagrams.

**Real-Time Unit Converter:** 22 engineering unit conversions built into 
the sidebar — pressure, temperature, flow rate, viscosity, heat transfer 
coefficients, power, and mass/force.

**Professional Export Tools:** PDF reports (jsPDF), Excel export (.xlsx), 
shareable calculation links, and print support.

**Calculation History:** Auto-saves last 50 calculations to localStorage 
with one-click replay to re-populate input fields.

---

## 🛠️ Technical Stack

- **Architecture:** Vanilla HTML + CSS + JavaScript (zero frameworks)
- **Charts:** Chart.js
- **Export:** jsPDF · SheetJS (xlsx)
- **Icons:** Phosphor Icons + Emoji
- **PWA:** Service Worker with offline caching · Web App Manifest
- **Storage:** localStorage for history, theme, and preferences
- **Sound:** Web Audio API — subtle HUD click/nav tones

---

## 📦 PWA Installation

ChemEFlow is installable as a native app on any device:

- **Android/Chrome:** Tap the install banner or the ⬇️ Install button in the sidebar
- **iOS/Safari:** Tap Share → Add to Home Screen
- **Desktop:** Click the ⊕ install icon in the browser address bar

Works fully **offline** once installed.
